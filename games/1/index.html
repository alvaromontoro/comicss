ABCnagrams

In this puzzle, you're given 26 scrambled clues, each representing a CSS property with one missing letter. Your task is to figure out the missing letter, add it back, and unscramble the result to reveal the correct CSS property. All missing letters are unique and don't repeat. In some cases, you may need to add a "-" to create the CSS property. The words and letters are not in alphabetical order. The first one has been solved as an example.
