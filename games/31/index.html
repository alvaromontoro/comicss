ABCnagrams

In this puzzle, you're given 26 scrambled clues, each representing an HTML tag with one missing letter. Your task is to figure out the missing letter, add it back, and unscramble the result to reveal the correct HTML element. All missing letters are unique and don't repeat. The words and letters are not in alphabetical order. Some of the HTML elements are deprecated, some others are experimental. Only the one for Z is not an actual HTML element but an attribute because there are no HTML tags containing the letter Z.
