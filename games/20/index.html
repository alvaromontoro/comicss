ABCnagrams

In this puzzle, you’re given 26 scrambled clues, each representing an HTML tag with one missing letter. Your task is to figure out the missing letter, add it back, and unscramble the result to reveal the correct HTML tag. All missing letters are unique and don't repeat. The words and letters are not in alphabetical order. Some of the HTML tags are deprecated, some others are experimental. The first one has been solved as an example (it's the only one that is not an HTML tag but a property.)
