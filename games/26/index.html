Crossword

Fill the crossword with the correct letters. Words will be read from left-to-right or from top-to-bottom (unless otherwise indicated). Special characters like hyphens or colons are omitted, so the answers will only contain letters (again, unless otherwise indicated).

Type to move from cell to cell, or use the tab key or arrow keys to navigate through the crossword. The puzzle will autosave as you fill it, so you can come back later if you want to take a break.

All related to CSS in some way. If you get stuck, you can find hints and the solution here. Good luck!

Note: dep. in a clue, indicates that the solution refers to a deprecated, obsolete, or removed feature. exp. in a clue indicates that the feature is experimental or it doesn't have full support.

Across

  1. Media type that matches printers or devices intended to simulate printed display
  2. Property that sets the color for user-interface controls
  3. Pseudo-element that acts as the last child
  4. Function that picks a value depending on a min, max, and preferred value
  5. Property that specifies the amount of space on one side of the element's content
  6. A display value

Down

  1. A white-space value for preformatted text that also collapses into new lines
  2. Cyan color
  3. Function to perform mathematical operations
  4. Psuedo-class that selects the top element of a tree representing the document
  5. Color that can be light, deep, or hot
