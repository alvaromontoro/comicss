ABCnagrams

In this puzzle, you're given 26 scrambled clues, each representing a CSS term with one missing letter. Your task is to figure out the missing letter, add it back, and unscramble the result to reveal the correct CSS property or value. All missing letters are unique and don't repeat. The words and letters are not in alphabetical order. Some of the CSS properties are deprecated, some others are experimental.
