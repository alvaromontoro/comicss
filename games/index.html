Games

comiCSS now has games! Over the past year, I've been working on creating a collection of CSS-based games (thanks everyone who tested them), and I'm excited to start adding them to the website on a regular basis. New games will be published every week on Wednesdays.

Some of these games are built entirely with CSS and will work even without JavaScript. Others will rely on JavaScript for additional functionality. In all cases, I'm taking a progressive enhancement approach to ensure a solid baseline experience while providing richer interactions where possible.
