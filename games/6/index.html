Code Breaker

These three knobs control a safebox lock. Each knob corresponds to a CSS word. To unlock the safebox, turn the knobs so that the correct starting letter of each word is at the top.

  1. Determine how many positions each knob needs to be turned.
  2. Specify the direction: C for clockwise or CC for counter-clockwise.
  3. Enter the number of positions and the direction in the inputs below each knob.
  4. Click "Check" to see if the safebox unlocks.

Note: Each CSS word can be read either left-to-right or right-to-left (whichever is the shortest distance). Words may contain dashes but they are not displayed on the knob.

Example:

R O B R E D

The solution for this knob would be 2-CC because the knob would need to be turned two letters counter-clockwise to unlock (BORDER).
